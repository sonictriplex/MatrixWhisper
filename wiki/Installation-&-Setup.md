# 🛠️ Installation & Setup

## Voraussetzungen

Stelle sicher, dass Python 3 und die benötigten Qt6‑Bibliotheken auf deinem Linux‑System installiert sind.

### Fedora / RedHat

```bash
sudo dnf install python3-pyqt6 python3-pyqt6-webengine
```

### Arch Linux / CachyOS / Manjaro

```bash
sudo pacman -S python-pyqt6 python-pyqt6-webengine
```

### Debian / Ubuntu

```bash
sudo apt install python3-pyqt6 python3-pyqt6-webengine
```

### openSUSE

```bash
sudo zypper install python3-pyqt6 python3-pyqt6-webengine
```

---

## Schnellstart

1. Repository klonen:
   ```bash
   git clone https://github.com/sonictriplex/MatrixWhisper.git
   cd MatrixWhisper
   ```

2. Anwendung starten:
   ```bash
   python3 matrixwhisper.py
   ```

---

## Autostart einrichten

MatrixWhisper kann automatisch mit dem Computer starten. Aktiviere die Option **Autostart** in den Einstellungen (Tab **Allgemein**). Die App erstellt dann eine `.desktop`-Datei in `~/.config/autostart/`.
