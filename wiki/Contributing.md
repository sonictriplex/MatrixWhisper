# 🤝 Mitwirken (Contributing)

## Übersicht

Beiträge sind willkommen! Egal ob Bugfixes, neue Funktionen, Übersetzungen oder Dokumentation – jede Hilfe zählt.

---

## Wie kann ich helfen?

### Bugs melden

Erstelle ein Issue im [GitHub-Repository](https://github.com/sonictriplex/MatrixWhisper/issues) mit:
- Beschreibung des Problems
- Schritten zur Reproduktion
- Erwartetem und tatsächlichem Verhalten
- Systeminformationen (Distribution, Python-Version, Qt-Version)

### Neue Funktionen vorschlagen

Erstelle ein Issue mit dem Label **Feature Request** und beschreibe:
- Was die Funktion tun soll
- Warum sie nützlich ist
- Mögliche Implementierungsideen

### Code beitragen

1. Forke das Repository.
2. Erstelle einen Branch für deine Änderungen.
3. Stelle sicher, dass der Code den bestehenden Stil einhält.
4. Erstelle einen Pull Request mit einer klaren Beschreibung.

### Übersetzungen beitragen

1. Füge einen neuen Eintrag im `TRANSLATIONS`-Dictionary in `matrixwhisper.py` hinzu.
2. Füge die Sprache zur `QComboBox` in `__init__` hinzu.
3. Erstelle einen Pull Request.

---

## Entwicklungsrichtlinien

- **Python 3.10+** – Verwende moderne Python-Features.
- **PyQt6** – Keine Qt5-Kompatibilität erforderlich.
- **PEP 8** – Halte dich an den Python-Code-Stil.
- **Keine externen Abhängigkeiten** – Nur PyQt6 und Standardbibliothek.
